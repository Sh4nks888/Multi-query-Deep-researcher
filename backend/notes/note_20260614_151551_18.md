# 任务 4: 学习路径

## 任务概览
- 任务主题：Agent技术学习路线调研
- 任务名称：学习路径
- 任务目标：整理从入门到进阶的系统学习路线，包括先修知识、课程、论文、项目实践和阶段性目标。
- 检索查询：AI Agent learning roadmap 2026 LLM Agent course paper project beginner advanced

## 来源概览
1. Edureka《Agentic AI Full Course 2026》：6小时级视频课程，覆盖 Agentic AI 概念、Agentic AI vs Generative AI、AI/深度学习/Transformer/LLM、LangChain、RAG、LLMOps、Prompt Engineering、NLP 等；页面显示 88,057 views、1,538 likes、频道 4.44M subscribers。
2. Maven《Build a Real AI Agent in 60 Minutes & 2026 Learning Roadmap》：强调构建与部署 production-ready AI Agent，而非 demo；页面显示 6,647 students；讲师 Ryan Ahmed 曾教授 510,000+ learners，覆盖 160 countries。
3. MachineLearningMastery《The Roadmap for Mastering Agentic AI in 2026》：提出从数学、编程、机器学习，到自治 Agent、Agent 架构、专业方向、部署与作品集的逐步路线。
4. Medium《The Realistic Guide to Mastering AI Agents in 2026》：标题信息显示主张以 6–9 个月路线掌握 AI Agents，从数学基础到生产系统部署。
5. Scaler《Agentic AI Self-Study Roadmap 2026》：给出 10 阶段自学路线，包含 Foundations、LLM、RAG、Agent Architecture、Multi-Agent、Tool Use、Memory & Autonomy、Evaluation & Safety、Deployment & MLOps、Portfolio Projects。

## 关键发现
1. Agent 学习路线应从基础能力开始，而不是直接跳到框架。来源普遍将 Python、ML、DL、NLP、GenAI、Prompt Engineering 视为前置基础；Edureka 明确列出这些先修项。
2. 学习目标正在从聊天机器人转向可执行工作流的自主系统。Scaler 强调 Agentic AI 能计划任务、使用工具、跨多步骤运行，适用于研究、分析、内部自动化、文档处理等复杂场景。
3. 系统路线适合拆成约 9–10 个月阶段化训练。Scaler 的 10 阶段路线可作为主骨架：基础、LLM、RAG、Agent 架构、多 Agent、工具集成、记忆自治、评估安全、部署 MLOps、作品集。
4. 进阶分水岭是 production-ready agent。Maven 明确强调 Build & Deploy a Production-Ready AI Agent (Not Just a Demo)，说明后期学习必须覆盖服务化、容器化、云部署、监控、评估、安全、权限和成本控制。
5. 最佳资源组合应是课程 + 论文/概念 + 项目实践。Edureka 适合快速建立知识地图；MachineLearningMastery 和 Scaler 适合路线拆解；Maven 适合生产部署导向；后续应补充经典 Agent/RAG/工具调用/多 Agent 论文与开源项目。

## 阶段性路线建议
- 入门阶段：Python、API、数据处理、基础 ML/DL/NLP、Prompt Engineering；目标是能调用 LLM API 并完成简单自动化脚本。
- 初级应用阶段：LLM 基础、Prompt 模式、RAG、向量数据库、文档问答；目标是能构建可用的知识库问答或研究助手。
- Agent 核心阶段：任务分解、规划、工具调用、函数调用、记忆、状态管理、LangChain/AutoGPT 等框架；目标是构建能处理多步任务的单 Agent。
- 进阶系统阶段：多 Agent 协作、长期记忆、反馈循环、评估体系、安全护栏、人机协同；目标是让 Agent 在复杂任务中稳定迭代。
- 工程落地阶段：FastAPI/Flask、Docker、云部署、日志监控、成本治理、权限控制、回归测试、作品集；目标是交付 production-ready capstone 项目。
