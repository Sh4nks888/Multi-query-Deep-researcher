# 任务 4: 性能与兼容性变化

# 任务概览
研究 Python 3.12 f-string 新特性中的性能提升、语法解析优化及向后兼容性问题。

# 来源概览
- What's New in Python 3.12 - F-Strings and Interpreter Changes (Andy Pearce)：强调重写解析器带来的错误报告增强和更灵活的语法。
- Python 3.14.5 官方文档：说明 f-string 限制被移除、PEP 693 发布计划、distutils 移除等。
- Medium 文章：提到对多核系统的性能提升。
- Black 格式化工具 Issue #3746：支持 PEP 701 语法形式化后的嵌套 f-string。
- Stack Overflow 问题：展示 3.12 f-string 中空格填充的用法。

# 初步关键发现（待完善）
1. 性能提升源于内部解析器重写，避免反复编译相同表达式。
2. 语法优化：支持嵌套引号、多行表达式、反斜杠等。
3. 兼容性：旧版本无法解析新语法，需注意部署环境。
4. 错误报告增强：内嵌表达式报错更精准。
5. 多核场景下的并行化潜力（需进一步确认）。
